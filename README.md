Informational website for ZEDX SMP. To be built with Nextjs and probably TailwindCSS. Possibly will integrate a DB in the future.

Problem domain - An up and coming Minecraft server with a lot of potential to rake in many players, needs a simple, sleek website where users can find out everything they need about the SMP without having to straight away join an active discord and get overwhelmed. The website needs to clearly state what the SMP is, what the rules are, how to join it and how to buy ranks with their descriptions.

User Stories:

As a User I want to easily navigate a simple website that tells me everything I need to know about ZEDX SMP.
As a User I want to see a list of rules, how to play, any available ranks with price and perks, and how to buy.

Developer stories:

As a Developer I want to design my website for a mobile first layout as I know that most people will view it from a mobile device.
As a Developer I want to integrate the IP into the website logo to maximise conversion to the game.
As a Developer I want to link to the creators multiple social media accounts.

Colour Palette: Black, Purple, White

We started making this with AI, so there is a whole lot to push straight away.
